Template Installation:

FTP Upload:
  1. Open up your FTP manager and connect to your hosting
  2. Browse to required directory (Normally public_html)
  3. Upload the files inside Lia folder.

Cpanel Upload:
  1. Open up your Cpanel File manager
  2. Enter public_html Folder
  3. Upload the files inside Lia folder.
